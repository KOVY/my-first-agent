# 📝 Příklady — Jak vyplnit SOUL.md a USER.md

Tady najdeš hotové příklady, které můžeš zkopírovat a upravit.

## Použití

1. Zkopíruj příklad do hlavní složky:
   ```bash
   cp examples/SOUL-example.md SOUL.md
   cp examples/USER-example.md USER.md
   ```
2. Otevři a uprav podle sebe
3. Spusť `qwen` a tvůj agent je ready!

## Soubory

- **SOUL-example.md** — agentka "Nova" — zvědavá, energická, kreativní
- **USER-example.md** — příklad uživatelského profilu
