# Skills — Dovednosti agenta

Skills rozšiřují co tvůj agent umí. Každý skill je složka s `SKILL.md` souborem.

## Přiložené skills

### 🔍 brave-search
Vyhledávání na webu přes Brave Search API. Zdarma do 2000 dotazů/měsíc.
- Registrace: https://brave.com/search/api/

### 🌤️ weather
Počasí pro jakékoliv místo přes wttr.in. Žádný API klíč, funguje hned.

## Jak přidat vlastní skill

1. Vytvoř složku ve `skills/`
2. Přidej `SKILL.md` s popisem co skill dělá
3. Přidej potřebné skripty
4. Agent si skill automaticky přečte když ho potřebuje
