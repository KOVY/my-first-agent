# Brave Search — Vyhledávání na webu

## Kdy použít
Když potřebuješ najít aktuální informace na internetu — novinky, dokumentaci, fakta.

## Jak to funguje
Používá Brave Search API. Zdarma do 2000 dotazů měsíčně.

## Nastavení
1. Zaregistruj se na https://brave.com/search/api/
2. Získej API klíč (zdarma)
3. Nastav proměnnou: `export BRAVE_API_KEY="tvůj-klíč"`

## Použití
Agent automaticky použije webové vyhledávání když potřebuje aktuální data.

Příklady dotazů:
- "Jaký je kurz koruny dnes?"
- "Najdi mi dokumentaci k Next.js 15"
- "Co se děje na trhu?"
